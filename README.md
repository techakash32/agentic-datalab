# Agentic DataLab

**Agentic DataLab** is an autonomous, multi-agent data science system that automates data ingestion, cleaning, modeling, evaluation, and natural-language reporting using LLMs.

##  Features
- Automated data ingestion and cleaning
- Model selection with scikit-learn
- Evaluation and LLM-powered report generation
- Streamlit dashboard
- CI/CD and Docker support

##  Tech Stack
Python 路 scikit-learn 路 pandas 路 Streamlit 路 OpenAI API 路 Docker
